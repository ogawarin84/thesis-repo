# 弦织体（String Weave Ontology）概念框架

以下是一个完全原创的概念框架，所有概念均为零基础创制，未借用任何现有理论。

## 1. 织点（Weave Point）
**记号**：`w_i`，其中 `i ∈ ℕ`
**属性**：
- **纹序** `o_i ∈ ℤ` —— 表示织点在抽象秩序轴上的位置，可为负。
- **韧度** `r_i ∈ ℝ⁺` —— 非负实数，表征织点抵抗变形的固有能力。

**定义**：织点是弦织体中最小的不可分实体，所有后续概念均基于织点集合 `W = {w_i}` 构建。

## 2. 弦线（String Line）
**记号**：`s_ij = (w_i, w_j)`，其中 `i ≠ j`
**属性**：
- **张力** `t_ij ∈ ℝ` —— 可正可负，正值表示拉伸，负值表示压缩。
- **相位** `p_ij ∈ [0, 2π)` —— 周期变量，表征弦线的振动偏移。

**定义**：弦线是连接两个不同织点的有序对，其存在性由**织网**决定。每条弦线携带张力和相位，两者共同决定织点间的相互作用强度与方向。

## 3. 织网（Weave Net）
**记号**：`𝒲 = (W, S)`，其中 `S ⊆ {s_ij ∣ w_i, w_j ∈ W, i ≠ j}`
**定义**：织网是由织点集 `W` 和弦线集 `S` 构成的图结构。织网的拓扑由**共振势**的动态演化决定，而非预先固定。

**生态依赖**：织网的边集 `S` 随**纹波**事件动态增减，新增弦线的初始张力与相位由其两端织点的当前状态计算得出。

## 4. 共振势（Resonance Potential）
**记号**：`R(w_i)`
**定义**：
```
R(w_i) = Σ (t_ij / (|o_i - o_j| + 1)) ⋅ sin(p_ij)
```
其中求和遍及所有以 `w_i` 为一端的弦线。

**解释**：共振势度量织点 `w_i` 受到的整体"谐和压力"。它同时依赖弦线的张力、相位差以及织点间的纹序距离。高共振势意味着该织点处于不稳定的激发状态，可能触发**纹波**。

**生态依赖**：共振势的计算依赖于织网中存在的弦线及其属性，而织网又受共振势反馈调节。

## 5. 纹波（Ripple）
**触发条件**：当 `R(w_i) > θ`（`θ` 为全局常数阈值）时，织点 `w_i` 释放一次纹波事件。
**效应**：对于每一条与 `w_i` 相连的弦线 `s_ij`，其张力和相位按如下规则更新：
```
t_ij ← t_ij + κ ⋅ R(w_i)
p_ij ← p_ij + λ ⋅ (r_i - r_j) mod 2π
```
其中 `κ, λ` 为正参数。此外，若更新后的 `|t_ij|` 超过某个临界值 `τ`，则从 `w_i` 向 `w_j` 方向生成一条新的弦线（若尚不存在），初始张力设为 `τ`，相位设为 `0`。

**生态依赖**：纹波改变弦线属性，进而改变共振势，从而可能引发连锁纹波。纹波也是织网增长的唯一机制。

## 6. 织序流（Weave Flow）
**定义**：整个系统的时间演化由以下常微分方程组控制（连续时间近似）：
```
d o_i / dt = Σ t_ij ⋅ cos(p_ij)
d r_i / dt = -μ r_i + ν Σ |t_ij|
```
其中 `μ, ν > 0` 为衰减与增益系数。

**解释**：纹序的变化率由弦线张力的投影之和决定，体现了织点沿秩序轴的漂移。韧度的变化包含自然衰减（第一项）和来自相连弦线总张力幅度的增强（第二项），形成负反馈回路。

**生态依赖**：织序流方程依赖于当前织网的状态（弦线集合及属性），而织网又因纹波事件随时间变化，因此这是一个耦合的离散-连续混合动力学系统。

## 7. 平衡态（Equilibrium）
**定义**：称织网 `𝒲` 处于平衡态，当且仅当对所有 `w_i ∈ W` 同时满足：
```
d o_i / dt = 0,  d r_i / dt = 0,  R(w_i) ≤ θ
```
此时系统不再产生纹波，织点纹序与韧度稳定，弦线属性恒定。

**生态依赖**：平衡态的存在性与稳定性取决于初始织网配置及参数 `θ, κ, λ, μ, ν, τ`。平衡态可视为织网的一种"吸引子"，而外界扰动（如随机增加新织点）可使系统脱离平衡态进入新的演化周期。

## 生态结构总结

```
织点 (w_i) ──拥有──> 纹序 o_i, 韧度 r_i
    │                    │
    │                    ▼
    ├──通过弦线连接──> 弦线 s_ij (张力 t_ij, 相位 p_ij)
    │                       │
    │                       ▼
    └──参与计算──> 共振势 R(w_i) ──触发条件──> 纹波事件
                         ▲                        │
                         │                        ▼
                         └──── 更新弦线属性 ────┘
                                                    │
                                                    ▼
                                             织序流方程 (do/dt, dr/dt)
                                                    │
                                                    ▼
                                             平衡态判定 (所有导数为0且R≤θ)
```

所有概念环环相扣：织点定义弦线，弦线构成织网，织网计算共振势，共振势触发纹波，纹波改变弦线并可能增加新弦线，同时织序流驱动纹序与韧度的连续变化，最终导向平衡态。没有任何一个概念独立于其他概念存在，形成了完整的自指生态。
